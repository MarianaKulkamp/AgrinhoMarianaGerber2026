# AgroTech 5.0

Este projeto é uma página estática moderna sobre o tema **AgroTech do Futuro** com foco em como tecnologias ajudam o produtor a produzir mais utilizando menos recursos naturais (Agro 5.0).

## O que o projeto faz

- Exibe um layout responsivo em três arquivos: `index.html`, `style.css` e `script.js`.
- Apresenta um **header** com logo fictícia e menu de navegação que rola a página para as seções correspondentes.
- Possui uma **seção hero** com título chamativo, descrição de valor e botões de chamada para ação.
- Mostra uma seção de **tecnologias** com cards interativos que explicam as soluções AgroTech:
  - Drones & Irrigação Inteligente
  - Inteligência Artificial no Campo
  - Sensores de Solo
  - Robótica & Automação
- Inclui um segmento de **simulador / gráfico** com contadores animados de recursos economizados e barras representando água, fertilizante e energia.
- Oferece um formulário de contato simples e uma seção de rodapé.

## Tecnologias utilizadas

- HTML sem frameworks
- CSS com layout responsivo usando Grid e Flexbox
- JavaScript puro para:
  - scroll suave ao clicar nos links do menu
  - interatividade de cards de tecnologia
  - animação de contadores quando a seção entra em tela

## Estrutura de arquivos

- `index.html` — marcação da página e seções principais
- `style.css` — definição de cores, tipografia, layout e responsividade
- `script.js` — comportamentos interativos do site
- `imagens/` — imagens usadas nos cards e na hero/logo

## Como usar

1. Abra o arquivo `index.html` no navegador.
2. Navegue pelas seções usando o menu superior.
3. Clique nos cards de tecnologia para ver a descrição extra.
4. Role até a seção de simulador para ver os contadores animados.

## Objetivo

Mostrar de forma visual e envolvente como o agro do futuro integra tecnologia e sustentabilidade para aumentar produtividade e reduzir consumo de recursos naturais.
